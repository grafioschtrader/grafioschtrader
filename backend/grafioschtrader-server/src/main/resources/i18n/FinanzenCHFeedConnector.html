Go to the website of finanzen.ch and find the desired security.
Where the stock exchange location can be selected, the appropriate one should be chosen.
Copy from the displayed URL only the name of the instrument including the abbreviation for the stock exchange. Normally these are separated by a slash
For example, "https://www.finanzen.ch/aktien/geberit-aktie/swx" becomes "geberit-aktie/swx".
This procedure can be used for both intraday and EOD prices.
