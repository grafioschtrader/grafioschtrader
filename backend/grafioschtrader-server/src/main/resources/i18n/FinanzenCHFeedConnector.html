[historical] Go to the website of finanzen.ch and find the desired security.
There, select the "Historisch" section and then change the start date and possibly the suggested stock exchange.
After that click "Historische Kurse anzeigen". Historical rates will only work if historical rates are available there.
Copy this URL without domain name and following ("/") as well as date range here.
For example, "https://www.finanzen.ch/kurse/historisch/swiss_re/swx/1.4.2021_20.5.2021" becomes "kurse/historisch/swiss_re/swx".
[intra] Go to the website of finanzen.ch and find the desired security.
Copy the entire URL without the domain name (financen.ch) and following slash.
For example, "http://www.finanzen.ch/aktien/UBS-Aktie" becomes "shares/UBS share".
