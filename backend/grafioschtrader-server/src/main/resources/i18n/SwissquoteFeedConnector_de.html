Bei den meisten Instrumenten kann die ISIN direkt verwendet werden.
Andernfalls kann auf swissquote.ch über einen Link auf das gewünschte Instrument diese Eingabe ermittelt werden.
In der Regel ist es die ISIN mit einer Sufix. Beispielweise die ISIN CH0009980894 für den Swiss Market Index (SMI) als CH0009980894_M9_CHF abgebildet.
