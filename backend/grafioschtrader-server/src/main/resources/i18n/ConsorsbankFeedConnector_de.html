Gehe auf die Webseite von Consorsbank.de Wähle "Suchen" und Eingabe der ISIN. Wird das Instrument gefunden. Es müsse die Werte von "id" und "rtExchangeCode" ermittelt werden. Dazu muss das Netzwerk der Entwicklertools im Webbrowser aktiviert werden. Mit der Auswahl der gewünschten Handelsplatzes bei der Kursangabe werden die Preise neu ermittelt. Für "LU0392496344" and der "SIX Schweiz" kann aus dem XHR-Aufruf die das "field=PriceV2" URL enthält ausgelesen werden. Der Werte von "id" ist in diesem Fall "_47633870" bzw. von "rtExchangeCode" "SWX". Die werden durch ein Komma "_47633870,SWX" getrennt eingeben.
