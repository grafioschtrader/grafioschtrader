Gehe auf die Webseite von wienerboerse.at und finde das gewünschte Wertpapier, für die Produktsuche wird die ISIN empfohlen.
Für die Eingabe wird der Wert von ID_NOTATION verlangt. Dieser kann der URL im Browser entnommen werden, nachdem Sie auf die Schaltfläche "Profi Chart" geklickt haben.
Beim Produkt mit der ISIN AT0000652011 ist beispielsweise 1534068 die korrekte Eingabe.
Für das einwandfreie Funktionieren des Konnektors muss die ISIN und Anlageklasse korrekt gesetzt sein.
