Dieser Anbieter läuft sicher einen Tag nach.
Gehe auf die Webseite von SWISS FUND DATA und finde das gewünschte Wertpapier
Die Menüpunt "Suche|Schnelle Suche" kann ein Instrument beispielsweise mit der ISIN erfolgen.
Am Schluss der URL sehen Sie eine Zahl, diese kann übernommen werden.
Bei Swisscanto (CH) Real Estate Fund Responsible IFCA (ISIN: CH0037430946) steht bei der URL "www.swissfunddata.ch/sfdpub/de/funds/show/13872" dabei wird 13872 übernommen.
