Gehe auf die Webseite von finance.yahoo.com und finde das gewünschte Wertpapier.
Die Eingabe entspricht dem Kürzel des Instruments. Dieses kann der URL oder direkt beim Names des Instruments entnommen werden.
^GSPC ist beispielsweise die Eingabe für den "S&P 500 (^GSPC)".
[historical] Nicht bei allen Instrumenten sind historischen Daten vorhanden, daher zuerst das Vorhandensein dieser Daten prüfen.
