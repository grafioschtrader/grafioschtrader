Für die Ermittlung von notationId muss Menüpunkt "Kurse|T&S/Historie" gewählt werden. Optional wählen der ensprechenden Börse und "Anzeigen" klicken.
Im geöffneten Fenster kann aus der URL die notationId gelesen werden. Für die Aktie Merck (DE0006599905) ist dies der Wert 158463, dieser wird in das Eingabefeld eintragen. Falls Menüpunkt "T&S/Historie" nicht vohanden:
Die Entwicklertools aktivieren und Reiter "Network" wählen.
