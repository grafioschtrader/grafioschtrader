Onvista liefert nur zwei Nachkommastellen, für Preise weniger 10 ist dies ungeeignet.

Aufonvista das entsprechende Wertpapier suchen.
Für die Ermittlung vonnotationId muss Menüpunkt "T&S/Historie" gewählt werden.
Optional wählen derensprechenden Börse und "Anzeigen" klicken.
Im geöffneten Fenster kann aus der URL die notationId gelesen werden. Für die Aktie Merck (DE0006599905) ist dies der Wert 158463, dieser wird in das Eingabefeld eintragen.
