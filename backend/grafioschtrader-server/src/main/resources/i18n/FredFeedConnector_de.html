Federal Reserve Economic Data (FRED), Endpoint für historische Beobachtungen. Wird verwendet, um den risikofreien Zinssatz pro Währung zu pflegen (genutzt z.B. von der Sharpe Ratio).
Ein kostenloser API-Schlüssel ist erforderlich, einzutragen über den bestehenden Admin-Dialog "Connector API Keys" (Provider-ID fred). Freie Stufe: 120 Anfragen/Minute, mehr als genug für einen Aufruf pro Serie und Tag.
Das Eingabefeld entspricht einer FRED-Serien-ID. Nachschlagen unter fred.stlouisfed.org.
Empfohlene Serien pro Währung (kurzfristig, für die Sharpe Ratio geeignet):
  USD → DGS3MO (3-Monats-Treasury, täglich)
  EUR → ECBESTRVOLWGTTRMDMNRT (€STR, täglich)
  GBP → IUDSOIA (SONIA, täglich)
  CHF → IR3TIB01CHM156N (3-Monats-Interbank, monatlich)
  JPY → IRSTCI01JPM156N (Call Rate, monatlich)
  Weitere OECD-Währungen → IR3TIB01[CC]M156N oder IRLTLT01[CC]M156N (10-Jahres, monatlich).
FRED liefert Zinssätze in Prozent; sie werden als Dezimalbruch in historyquote.close gespeichert (z.B. 4.32% → 0.0432). Fehlende Werte (FRED kodiert sie als ".") werden übersprungen.
