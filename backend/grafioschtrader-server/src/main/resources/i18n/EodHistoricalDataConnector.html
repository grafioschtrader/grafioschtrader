The free version of EOD Historical Data should not be used due to its limitations.
On the main page of EOD Historical Data you can find the corresponding URL extension with the symbol of the instrument. This usually consists of the symbol of the instrument and the abbreviation of the stock exchange with a dot in between.
For currency pairs, consult "List of Supported Forex Currencies". The URL extension for cryptocurrencies can be found at "List of Supported CRYPTO Currencies".
