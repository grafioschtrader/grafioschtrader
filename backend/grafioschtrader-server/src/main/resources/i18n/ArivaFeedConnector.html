Ariva delivers the data only from the 2004-01-04.

On ariva.de search for the corresponding security. To determine the two values secu, boerse_id and possibly also currency, the network of developer tools must be activated in the web browser.
The following example shows the historical prices of the Richemont share. Then press the button "Download" via "Kurse|Historische Kurse". Via "Request URL" the section after "secu=" including "boerse_id=123" and possible currency "&currency=CHF". copy. For Richemont, the following value is determined "101396719&boerse_id=123&currency=CHF".
