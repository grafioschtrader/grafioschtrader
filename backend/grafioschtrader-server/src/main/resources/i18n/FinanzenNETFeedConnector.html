[historical] Go to the website of finanzen.net and find the security you want.
There, select the "Historisch" section and then change the start date and possibly the suggested stock exchange.
After that click the button "Historische Kurse anzeigen".
Copy the displayed URL without domain name and then.
For example, "https://www.finanzen.net/historische-kurse/daimler" becomes "historical-prices/daimler".
[intra] Go to the website of finanzen.net and find the security you want.
Select the corresponding stock exchange.
For the Daimler share on the XETRA stock exchange, for example, "aktien/daimler-aktie@stBoerse_XETRA" of "https://www.finanzen.net/aktien/daimler-aktie@stBoerse_XETRA" is entered here.
