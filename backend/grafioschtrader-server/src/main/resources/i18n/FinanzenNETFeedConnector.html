[historical]

Finanzen.net can no longer serve as ETF data provider since 12/28/2021!

Go to the website of finanzen.net and find the security you want.
There, select the "Historisch" section and then change the start date and possibly the suggested stock exchange.
After that click the button "Historische Kurse anzeigen".
Copy the displayed URL without domain name and then.
For example, "https://www.finanzen.net/historische-kurse/daimler" becomes "historical-prices/daimler".
Commodites not investable or direct investment
The URL must also be supplemented with the URL determined via the developer tools. For example, for the oil price WTI in USD with "|oelpreis/USD" because the request URL is for example "https://www.finanzen.net/Ajax/CommodityController_HistoricPriceList/oelpreis/USD/9.4.2021_28.5.2021". [intra] Go to the website of finanzen.net and find the security you want.
Select the corresponding stock exchange.
For the Daimler share on the XETRA stock exchange, for example, "aktien/daimler-aktie@stBoerse_XETRA" of "https://www.finanzen.net/aktien/daimler-aktie@stBoerse_XETRA" is entered here.
