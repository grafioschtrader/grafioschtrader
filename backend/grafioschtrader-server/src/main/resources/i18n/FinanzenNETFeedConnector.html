[intra] Go to the website of finanzen.net and find the security you want.
Select the corresponding stock exchange.
For the Daimler share on the XETRA stock exchange, for example, "aktien/daimler-aktie@stBoerse_XETRA" of "https://www.finanzen.net/aktien/daimler-aktie@stBoerse_XETRA" is entered here.
Bei Devisen das Suffix des Währungspaares hinzfügen, aus "https://www.finanzen.net/devisen/schweizer_franken-euro-kurs" wird "schweizer_franken-euro-kurs".
