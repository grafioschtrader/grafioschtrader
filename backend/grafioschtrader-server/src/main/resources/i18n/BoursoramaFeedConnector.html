For the determination of the value symbol, the network of developer tools must be activated in the web browser.
After that click on one of the offered time periods in the chart, this will trigger an Ajax call. From this request URL you can get the symbol. For example, the symbol "1rPCAC is determined for CAC 40, which must be entered.
