Die freie Version von EOD Historical Data solle aufgrund der Einschränkungen nicht gentutzt werden.
Auf der Hauptseite von EOD Historical Data können Sie mit Symbol des Instrumentes die entsprechende URL-Erweiterung finden. Diese besteht normalerweise aus dem Symbol des Instrumentes und dem Kürzel des Handelsplatzes mit einem Punkt dazwischen.
Für die Währungspaare konsultieren Sie die Seite "List of Supported Forex Currencies". Die URL-Erweiterung für Kryptowährungen kann unter "List of Supported CRYPTO Currencies" gefunden werden.
