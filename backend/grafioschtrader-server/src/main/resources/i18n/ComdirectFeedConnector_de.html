Comdirect produziert viele Timeout- oder Handshake-Errors, daher könnte dieser in einer späteren GT-Versionen entfernt werden. Nur eingeschränkt benutzen!
Gehe auf die Webseite von comdirect und finde das gewünschte Wertpapier
Oftmals muss das Instrument noch über die "Trefferliste" nochmals selektiert werden, um eine bereinigte URL zu bekommen, d.h. ohne "SEARCH_REDIRECT" usw.
Danach den Teil nach "...inf/" von der URL übernehmen. Beispielsweise wird "aktien/volkswagen-aktie-DE0007664039" von "https://www.comdirect.de/inf/aktien/volkswagen-aktie-DE0007664039" übernommen.
