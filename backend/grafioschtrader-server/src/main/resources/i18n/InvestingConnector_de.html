[historical] Zuerst das entsprechende Instrument (Wahrung, ETF, CFD, usw.) bei Investing finden.
Für die Ermittlung des gesuchten Symbols muss die Funktion Network der Entwicklertools im Webbrowser aktiviert werden. Zusätzlich sollte nach XHR-Aufrufen gefiltert werden.
Danach klicken Sie Chart und suchen in den Entwicklertools die Request URL mit https://tvc4.investing.com oder https://tvc6.investing.com. Aus dieser kann der symbol-Parameter gelesen werden.
Beispielweise wird für den Index S&P 500 (SPX) das Symbol "166" ermittelt, dieser Wert wird danach eingetragen
[intra] Das entsprechende Instrument (Wahrung, ETF, CFD, usw.) bei Investing finden.
Danach den Teil ohne "https://www.investing.com/" von der URL übernehmen. Beispielsweise wird aus "https://www.investing.com/equities/dropbox-inc" für die Aktie Dropbox Inc (DBX) nur "equities/dropbox-inc" übernommen.
