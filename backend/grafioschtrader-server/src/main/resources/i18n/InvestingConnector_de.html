[historical] Zuerst das entsprechende Instrument (Währung, ETF, CFD, usw.) bei Investing finden.
Danach "Historical Data" klicken, bspw. S&P 500 (SPX)
Für die Ermittlung der beiden Werte curr_id , smlID muss das Network der Entwicklertools im Webbrowser aktiviert werden.
Mit der Eingabe eines Datumbereichs wird ein XHR request https://www.investing.com/instruments/HistoricalDataAjax ausgelöst.
Unter "Form Data" kann curr_id und smlID ausgelesen werden, beispielsweise bspw. curr_id: 166 und smlID: 2030167 für dem S&P 500.
Nachflogend kann die URL ohne Domain Name des Instruments, curr_id und smlID durch Kommas getrennt eingeben werden, bspw. "indices/us-spx-500,166,2030167 für den S&P 500".
