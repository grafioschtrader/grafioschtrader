Search or select the corresponding security on onvista
Click on the button "Alle Kurse" or. "Historische Kurse" to get to the button "Download as CSV".
Activate the developer tools of the web browser and select the menu item "Network".
Press the "Download als CSV". We take a certain part of the request URL from the developer tools of the network.
For example, for the stock Amazon at the trading place "Nasdaq" we get the following URL request: "https://api.onvista.de/api/v1/instruments/STOCK/90929/eod_history?idNotation=252992&range=M1&startDate=2022-06-28" we need the part between "https://api.onvista.de/api/v1/instruments/" and "&range=M1&startDate=2022-06-28"". Therefore we enter "STOCK/90929/eod_history?idNotation=252992".
