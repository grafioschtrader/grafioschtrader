Onvista provides only two decimal places, for prices less than 10 this is unsuitable.

On onvista search for the corresponding security. For the determination of notationId menu item "T&S/History" must be selected. Optionally select the corresponding stock exchange and click "Anzeigen". In the opened window the notationId can be read from the URL. For the share Merck (DE0006599905) this is the value 158463, which will be entered in the input field.
