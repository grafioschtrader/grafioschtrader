Gehe auf die Webseite von six-swiss-exchange.com und finde das gewünschte Wertpapier, für die Produktsuche wird die ISIN empfohlen.
Danach kann die Eingabe der URL entnommen werden, dieser entspricht dem Wert der id, diese enspricht der ISIN mit einer Suffix.
Beim Produkt mit der ISIN IE00BYXYYJ35 ist beispielsweise IE00BYXYYJ35CHF4 die korrekte Eingabe.
Dieser Konnektor verlangt die korrekte Angabe der Anlageklasse.
