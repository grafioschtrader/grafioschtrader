Boursorama hat in den historischen Kurse selten hohe Kursausschläge, bitte in Tageskurse kontrollieren und solche Ausschläge löschen. Zudem werden maximal 20 Jahre historische Daten geliefert, dies ist nicht immer ausreichend.
Gehe auf die Webseite von Boursorama und finde das gewünschte Wertpapier.
Für die Ermittlung des Wertes symbol, muss das Netzwerk der Entwicklertools im Webbrowser aktiviert werden.
Danach klicken Sie auf einen der angebotenen Zeiträume im Chart, dies wird einen Ajax-Aufruf auslösen. Aus dieser Anfrage-URL kann das Symbol entnommen werden.Beispielsweise wird für CAC 40 das Symbol "1rPCAC" ermittelt, welches eingetragen werden muss.
