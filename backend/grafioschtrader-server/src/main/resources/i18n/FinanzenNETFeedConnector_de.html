[historical] Gehe auf die Webseite von finanzen.net und finde das gewünschte Wertpapier.
Wähle dort die Rubrik "Historisch" und verändere danach das Startdatum und möglicherweise die vorgeschlagene Börse.
Danach clickst du die Schaltfläche "Historische Kurse anzeigen".
Kopiere die angezeige URL ohne Domainname und anschliessendes.
Aus "https://www.finanzen.net/historische-kurse/daimler" wird beispielweise "historische-kurse/daimler". [intra] Gehe auf die Webseite von Finanzen.net und finde das gewünschte Wertpapier.
Wähle den ensprechenden Börsenplatz aus.
Für die Daimler Aktie an der Börse XETRA wird beispielsweise "aktien/daimler-aktie@stBoerse_XETRA" von "https://www.finanzen.net/aktien/daimler-aktie@stBoerse_XETRA" hier eingetragen.
