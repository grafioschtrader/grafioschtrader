[historical] Gehe auf die Webseite von finanzen.ch und finde das gewünschte Wertpapier.
Wähle dort die Rubrik "Historisch" und verändere danach das Startdatum und möglicherweise die vorgeschlagene Börse.
Danach clickst du "Historische Kurse anzeigen".
Kopiere diese URL ohne Domainname und anschliessendes ("/") sowie Datumsbereich hierher.
Aus "https://www.finanzen.ch/kurse/historisch/swiss_re/swx/1.4.2021_20.5.2021" wird beispielweise "kurse/historisch/swiss_re/swx".
[intra] Gehe auf die Webseite von finanzen.ch und finde das gewünschte Wertpapier.
Kopiere die gesamte URL ohne den Domainname (financen.ch) und nachfolgendem Schrägstrich.
Aus "http://www.finanzen.ch/aktien/UBS-Aktie" wird beispielsweise "aktien/UBS-Aktie".
