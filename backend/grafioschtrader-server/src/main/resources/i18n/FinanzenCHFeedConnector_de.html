Gehe auf die Webseite von finanzen.ch und finde das gewünschte Wertpapier.
Wo der Börsenplatz ausgewählt werden kann, sollte der entsprechende gewählt werden.
Kopiere von der angezeigten URL nur den Namen des Instruments inklusive das Kürzel für die Börse. Normalerweise sind diese durch den Schrägstrich getrennt
Aus "https://www.finanzen.ch/aktien/geberit-aktie/swx" wird beispielsweise "geberit-aktie/swx".
Dieses Prozedere kann sowohl für Intraday- als auch EOD-Kurse angewendet werden.
