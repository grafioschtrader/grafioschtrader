Federal Reserve Economic Data (FRED) historical observations endpoint. Used to populate the risk-free interest rate per currency (consumed e.g. by the Sharpe ratio).
Requires a free API key, registered under the existing "Connector API Keys" admin screen (provider id fred). Free tier: 120 requests/minute, more than enough for one call per series per day.
The input field corresponds to a FRED series id. Lookup at fred.stlouisfed.org.
Recommended series per currency (short-term, suitable for Sharpe):
  USD → DGS3MO (3-month Treasury, daily)
  EUR → ECBESTRVOLWGTTRMDMNRT (€STR, daily)
  GBP → IUDSOIA (SONIA, daily)
  CHF → IR3TIB01CHM156N (3-month interbank, monthly)
  JPY → IRSTCI01JPM156N (call rate, monthly)
  Other OECD currencies → IR3TIB01[CC]M156N or IRLTLT01[CC]M156N (10-year, monthly).
Rates are returned as percent by FRED and stored as decimal fractions in historyquote.close (e.g. 4.32% → 0.0432). Missing observations (encoded "." by FRED) are skipped.
